# tova-legal

Static legal pages for **Tova通话** (iOS / Android translation app), hosted
on GitHub Pages.

- Live URL: https://pinyinpals.github.io/tova-legal/
- Source of truth for `AppUrls.privacyPolicy` in the Flutter app.

## Updating

Edit `index.html` on `main`. GitHub Pages redeploys within ~1 minute.

The in-app `PrivacyPolicyScreen` must be kept in sync when material changes
happen — update the "Last updated" date in both places.
